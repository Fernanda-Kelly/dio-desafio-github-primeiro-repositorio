O que é lógica? Coerência de raciocínio de ideias.

O que é lógica de programação? Apenas contextualizar a lógica na programação de computadores.

Pseudocódigo -> É uma forma genérica de escrever um algoritmo, utilizando
uma linguagem simples.

Fluxograma -> É uma ferramenta utilizada para representar graficamente o
algoritmo.

Variáveis -> Na programação, uma variável é um objeto (uma posição,
frequentemente localizada na memória) capaz de reter e representar um valor 
ou expressão.

Constante -> São valores imutáveis e não são alterados durante a vida útil
do programa.

O que é linguagem de programação? Sua função é servir de um meio de comunicação
entre computadores e humanos.

Linguagens:

-> ALTO NÍVEL = Cuja sintaxe se aproxima mais da nossa linguagem e se distanciam
mais da linguagem da máquina. 

Ex: C, PHP, JS, C#, C++, PYTHON e etc. 

-> BAIXO NÍVEL = É aquela que se aproxima mais da longuagem da máquina, precisando 
ter conhecimento de arquitetura do computador.

Ex: Assembly.

-> COMPILADAS = É uma linguagem em que o código fonte é executado diretamente pelo
Sistema Operacional ou pelo processador, após ser traduzido por meio de um processo 
chamado compilação.

Ex: C#, VB, DELPH, C++.

-> INTERPRETADAS = É uma linguagem em que o código fonte é executado por um programa
de computador chamado interpretador que em seguida é executado pelo Sistema Operacional
ou pelo processador.

Ex: JS, PHP, PYTHON.
